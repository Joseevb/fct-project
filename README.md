# Proyecto FCT - Desarrollo de Aplicacioens Web Dual

## Descripción

Este proyecto consiste en una página web para una empresa de maquillaje. El funcionamiento de la página consiste en la posibilidad de realizar funciones básicas para una empresa de maquillaje:

- Reservas de citas de maquillajes.
- Inscripción en cursos.
- Compra/venta de productos.

## Tecnologías

- Frontend: React, TypeScript, Tailwind CSS, Vite, Shadcn UI
- Backend: Java, Spring Boot
- Base de datos: MariaDB, H2

## Requisitos para la ejecución

- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/)

## Instalación

1. Clonar el repositorio
2. Ejecutar los comandos:

```bash
docker compose up -d --build
```

3. Abrir la aplicación en el navegador:

```bash
http://localhost
```

4. Para cerrar la aplicación, ejecutar los siguientes comandos:

```bash
docker compose down
```
